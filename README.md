# linkedList_C
Basic linked list implementation in C  

###Operations described and supported in linkedList.h:

- createLinkedList
- createNode
- appendToLinkedList
- dislayLinkedList
- addNodeToPosition
- deleteNodeFromPosition
- linkedListLength
- updateNodeInPosition
